HTML Formatting, Headings and Paragraph

Formatting

Formatting were designed to display special types of text. for example:

The Bold tag : defines bold text, without any extra importance.

The strong tag : defines text with strong importance

The Italic tag: is often used to indicate a technical term, a phrase from another language, a thought, a ship name, etc.

Emphasis tag: defines emphasized text. The content inside is typically displayed in italic

Marked tag: defines text that should be marked or highlighted.

Insert tag: defines text that should be marked or highlighted:

Delete tag: defines text that should be marked or highlighted:

Subscript tag: defines superscript text. Superscript text appears half a character above the normal line. e.g C 6 H 12 O 6

Superscript tag: lement defines superscript text. Superscript text appears half a character above the normal line. e.g X 2 Y 3= 9

Headings Tags

Heading or h tags are used as titles or subtitles that you want to display on a webpage. Headings are defined with h1 to h6.

h1 defines the least important heading and h6 defines the least important heading. i.e h1 having the biggest font size and h6 having the smallest font size
